## 프로젝트 소개
Unity로 약 1달간 개발한 3D RPG 인디게임입니다.
탑뷰 3인칭으로 진행되며 던전과 마을 맵이 있습니다.  
마을에서는 음식 제작과 무기 강화를 통해 던전에서의 전투를 준비하고(구현X), 던전에서의 플레이어는 강화를 토대로 적들과 싸우게 됩니다.
